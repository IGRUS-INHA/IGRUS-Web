이 폴더는 학기별 동아리 회원 명단 아카이빙 기능 관련 문서를 포함합니다.

## 기능 개요

유저 테이블과 별개로 학기별 회원 명단의 스냅샷을 생성하고 보관하는 기능입니다.

### 주요 특징

- 관리자가 개별 회원을 체크하여 아카이브에 추가/제외
- Soft Delete 적용으로 데이터 복구 가능
- 빈 아카이브만 삭제 가능 (데이터 보호)
- CSV 형식 내보내기 지원
